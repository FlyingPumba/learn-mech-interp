# Learn Mechanistic Interpretability

An open-source textbook on mechanistic interpretability. 47 articles covering transformer internals, interpretability techniques, and frontier research, organized as a structured learning path from foundations to open problems.

**[Start reading &rarr;](https://learnmechinterp.com/)**

## Table of contents

13 blocks, progressing from foundational concepts to open research questions:

| # | Block | Articles |
|---|-------|----------|
| 1 | Transformer Foundations | Prerequisites, Architecture, Attention, QK/OV Circuits, Composition |
| 2 | Interpretability Fundamentals | What is Interpretability?, Linear Representation, Superposition |
| 3 | Basic Techniques | Logit Lens, Direct Logit Attribution, Reading Attention Patterns |
| 4 | Causal Interventions | Activation Patching, Attribution & Path Patching |
| 5 | Probing | Probing Classifiers, CAA, LAT, Attention Probes |
| 6 | Steering | Addition, Ablation, Representation Control, Function Vectors |
| 7 | Model Editing | Concept Erasure with LEACE |
| 8 | Superposition & Feature Extraction | SAEs, Feature Dashboards, Scaling Monosemanticity, SAE Variants, Feature Geometry |
| 9 | Hidden State Decoding | Decoding Vectors, Patchscopes, SelfIE, Training Explanations, LatentQA, Activation Oracles |
| 10 | Circuit Finding | Induction Heads, IOI Circuit, Circuit Evaluation, Transcoders, Circuit Tracing, Crosscoders, Universality, Multimodal MI |
| 11 | MI for AI Safety | Refusal Direction, Sleeper Agents, Deception Detection, Safety Mechanisms, Honest Limitations |
| 12 | Tools | TransformerLens, nnsight & nnterp |
| 13 | Open Problems & Field Assessment | Open Problems, Field Assessment, Future of MI, Course Synthesis |
